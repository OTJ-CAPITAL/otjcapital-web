# otjcapital.com

Marketing and investor website for **OTJ Capital** — an AI-native quantitative hedge fund headquartered in Nairobi, Kenya.

---

## What this site is

The primary public face of the fund. Communicates the investment thesis, trading strategies, risk model, and fundraising roadmap to prospective investors and partners.

**Sections:**
- Hero — fund positioning and key stats (4 asset classes, 42 strategies, $100M target)
- About — the Africa opportunity and why quant infrastructure is missing
- Strategies — EMA Momentum, Bollinger Mean Reversion, Volatility Breakout, Funding Rate
- Risk Management — the 6 non-negotiable risk rules
- Roadmap — seed → first close → preseed → Series A → $100M
- CTA — mailto:invest@otjcapital.com

---

## Stack

| Layer | Technology |
|---|---|
| Framework | Next.js 14, TypeScript |
| Styling | Tailwind CSS |
| Deployment | Vercel |
| Domain | otjcapital.com |

---

## Development

```bash
npm install
npm run dev
# → http://localhost:3000
```

---

## Deploy

1. Connect `OTJ-CAPITAL/otjcapital-web` to Vercel
2. No environment variables required
3. Set custom domain: `otjcapital.com`
4. Every push to `main` auto-deploys
